# 📦 WB Arbitrage Parser & Alert System

**Автоматический поиск арбитражных связок на Wildberries с уведомлениями в Telegram**

[![Python](https://img.shields.io/badge/Python-3.9+-blue)](https://www.python.org/)

---

## 🎯 О проекте

Парсер товаров Wildberries для поиска арбитражных возможностей: товаров, где кешбэк за отзыв превышает цену товара (`cashback > price`). Система автоматически собирает данные, фильтрует выгодные предложения и отправляет уведомления в Telegram-канал.

**Цель:** Автоматизировать поиск прибыльных товаров для перепродажи или заработка на кешбэке.

---

## ✨ Возможности

### 📥 Сбор данных
- ✅ Парсинг 10 000+ товарных карточек с Wildberries
- ✅ Сбор динамических данных: цена, отзывы, условия кешбэка
- ✅ Обработка изменений цен в реальном времени

### 🔍 Фильтрация и анализ
- ✅ Автоматическая фильтрация арбитражных связок: `cashback > price`
- ✅ Расчет потенциальной прибыли для каждого товара
- ✅ Экспорт результатов в Excel/CSV для дальнейшего анализа

### 🔔 Уведомления
- ✅ Автоматическая отправка алертов в Telegram-канал
- ✅ Стабильная работа: 100+ релевантных офферов в день
- ✅ Форматированные сообщения с ключевыми параметрами товара

### ⚙️ Автономная работа
- ✅ Планировщик задач для регулярного запуска
- ✅ Обработка HTTP-ошибок и повторные попытки запросов
- ✅ Логирование для отслеживания работы и диагностики
